# Ops reporting pack — trial mapping

This pack activates the following workflow trials (defined in the repo's trial_registry):

| Trial ID                  | Purpose                                                                 |
|---------------------------|-------------------------------------------------------------------------|
| ops_summarize_reporting   | Summarize user's recurring reporting workflow; suggest weekly status    |
| ops_scaffold_status       | Scaffold weekly status report package in user's style                  |
| ops_next_steps            | Recommend next steps based on projects and routines                    |

Release run and pilot use these when the pack is active (role=ops or pack installed and activated).
